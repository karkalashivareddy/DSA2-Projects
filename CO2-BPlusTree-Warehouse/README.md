# B+ Tree Warehouse Product Management

## Project Description

This project demonstrates the implementation of a B+ Tree for warehouse product indexing and range query optimization.

## Features

* Product Indexing
* Sequential Traversal
* Efficient Range Queries
* Sorted Data Storage

## Technologies Used

* Java
* Data Structures

## Concepts Used

* B+ Tree
* Database Indexing
* Range Queries

## Advantages

* Fast Search Operations
* Efficient Disk Access
* Better Range Query Performance

## Time Complexity

* Search: O(log n)
* Insert: O(log n)

## Output

Displays:

* Product Records
* Range Query Results
* Sorted Traversal

# Graph Traffic Optimization System

## Project Description

This project implements graph traversal and Minimum Spanning Tree algorithms for traffic network optimization.

## Features

* BFS Traversal
* DFS Traversal
* Prim's Algorithm
* Kruskal's Algorithm

## Technologies Used

* Java
* Graph Algorithms

## Applications

* Traffic Optimization
* Network Planning
* Smart City Systems

## Algorithms Used

* BFS
* DFS
* Prim's MST
* Kruskal's MST

## Output

Displays:

* Graph Traversal
* Minimum Spanning Tree
* Minimum Cost

# AVL Library Management System

## Project Description

This project implements an AVL Tree for managing library book records efficiently. The AVL Tree maintains balanced height to ensure fast insertion, deletion, and searching operations.

## Features

* Insert Book IDs
* Search Book IDs
* Delete Book IDs
* Automatic AVL Rotations
* Balanced Tree Maintenance

## Technologies Used

* Java
* Data Structures and Algorithms

## Algorithms Used

* AVL Tree
* LL Rotation
* RR Rotation
* LR Rotation
* RL Rotation

## Time Complexity

* Insertion: O(log n)
* Deletion: O(log n)
* Search: O(log n)

## Output

The program displays:

* Inserted Book IDs
* Traversal Output
* Search Results
* Deletion Results
